# 💰 Expense Analyzer Baba

An AI-powered personal expense tracking application built using **Streamlit** and **Groq LLM (LLaMA 3.3 70B)**.

This app automatically categorizes your expenses using AI and provides intelligent financial insights with visual analytics.

## ✨ Features

- 🤖 AI-based automatic expense categorization
- 📊 Interactive visualizations (Pie, Bar & Trend charts)
- 📈 Spending trend analysis over time
- 💡 AI-generated financial insights & budgeting recommendations
- 💾 CSV-based local data persistence
- 🔐 Secure API key handling using environment variables

## 🛠️ Tech Stack

- **Python**
- **Streamlit**
- **Pandas**
- **Matplotlib**
- **Groq API**
- **LLaMA 3.3 70B Versatile Model**

---

## 🧠 How It Works

1. User enters expense amount and description.
2. Groq LLM predicts the most relevant category.
3. Data is stored locally in a CSV file.
4. Visual charts summarize spending patterns.
5. AI analyzes overall behavior and suggests budgeting improvements.
